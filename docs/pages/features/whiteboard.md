# Whiteboard

![Whiteboard](/images/whiteboard.webp)

**Whiteboard** opens an independent, clean canvas for freeform drawing — separate from the overlay that draws on top of apps.

**Default shortcut:** `Ctrl + 4`

## How It Works

Unlike [Live Draw](/features/live-draw) (which draws on a transparent layer over your screen), Whiteboard opens a **dedicated opaque window** with a solid background. It uses the same drawing engine but provides a blank canvas for sketching ideas.

## Features

### Full Drawing Tools

The whiteboard includes the same complete drawing toolset as Live Draw:

- Pen, marker, shapes (rectangle, ellipse, line, arrow)
- Text placement and editing
- Eraser and select tool
- All color, gradient, and stroke width options
- Undo / Redo

### Grid

Toggle an optional **grid overlay** on the whiteboard background.

### Infinite Canvas

The whiteboard supports **infinite canvas** — scroll and pan to access unlimited drawing space beyond the visible window area.
- To pan, use the mouse wheel or trackpad (with 2 fingers).
- To zoom, hold down the `Ctrl` key and use the mouse wheel or trackpad (with 2 fingers).

### Export

Save your whiteboard content as a **PNG image** file. This captures the current visible canvas with all drawings included. Saved files are stored in your downloads folder.

## Toolbar

The whiteboard has its own floating toolbar, independent of the overlay toolbar. Its position is saved separately and persists between sessions.

## Use Cases

- **Brainstorming**: Sketch ideas during meetings without switching to another app
- **Teaching**: Draw anything on a clean surface
- **Quick notes**: Jot down visual notes during presentations
- **Explaining concepts**: Use as a scratchpad during screen shares
