# Roadmap

Vynta is a side project of mine that I'm working on in my free time. So, I'm planning to release a new version every time I have enough time to work on it. They still many features that I want to add to Vynta.

## Currently in Development

- **Screenshot with Annotations:** Capture a screenshot of your screen with overlay annotations included, creating an image that shows both the underlying content and your drawings.

## Planned Feature

- **Screen Recording:** Record your screen along with all live annotations, cursor highlights, and zoom effects directly from the application. This will be ideal for creating educational videos, tutorials, and remote demonstrations.
- **Advanced Export Options:**
    - **Export Profiles:** Configurable settings for image and video exports, including quality, resolution, and format.
    - **Cloud Integration:** Direct upload capability to common platforms or cloud storage services.
- **Extended Toolset:**
    - **More Shapes and Drawing Tools:** Expand the drawing toolset with additional primitive shapes, custom arrows, and stamps.
    - **Notes and diagrams:** Add the ability to add notes and diagrams to the canvas.

## Maybe
- **Cross-platform Capabilities:** While currently highly optimized for providing a native Windows experience, Vynta's Tauri-based architecture allows us to explore future support for macOS and Linux platforms.
